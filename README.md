# SwiftShop - E-Commerce Product Catalog

A responsive e-commerce product catalog built with React, Vite, CSS custom properties, and standard CSS stylesheets. Features custom category and price range filters, a responsive filter drawer for mobile viewports, live search, sorting, and toggleable light/dark modes.

## Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed (recommended version LTS 18+ or 20+).

## Getting Started

Follow these steps to run the project locally:

### 1. Install Dependencies

Install all package dependencies using npm:

```bash
npm install
```

### 2. Run the Development Server

Start the local development server:

```bash
npm run dev
```

Once started, open [http://localhost:5173](http://localhost:5173) in your web browser.

---

## Additional Commands

### Build for Production

Compile and bundle the application into the static `dist` folder:

```bash
npm run build
```

### Run Linter

Check for code syntax issues and verify code quality:

```bash
npm run lint
```

### Preview Production Build

Serve the compiled production assets locally to test the build:

```bash
npm run preview
```
